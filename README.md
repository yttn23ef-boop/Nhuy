Project details:
CampusBook is a decentralized marketplace where students can buy and sell second-hand textbooks using a custom campus token. The platform uses blockchain smart contracts to secure payments, reduce transaction costs, and build trust between buyers and sellers. Students can list books, make instant payments, and confirm delivery through a simple mobile interface.

Vision:
To create a trusted campus economy where students can access affordable learning materials, reduce waste, and experience real blockchain payments in daily student life.
